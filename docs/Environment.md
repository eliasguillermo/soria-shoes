# Environment Docs
- Configure a `.env` file in your local project and add it to `.gitignore`
- Set the following Env variables:
    FIREBASE_API_KEY = 
    FIREBASE_AUTH_DOMAIN = 
    FIREBASE_DATABASE_URL = 
    FIREBASE_PROJECT_ID = 
    FIREBASE_STORAGE_BUCKET = "
    FIREBASE_MESSAGING_SENDER_ID = 
    FIREBASE_APP_ID = 

## Run in local
- Run `npm run dev`
